# 简介
### 灵感来源于某服务器5周目时的一个插件但是断更了 遂写之当java练手 欢迎批判（

就是一个简简单单可以@人的东西

在聊天栏中@+用户名或直接输入用户名即可在公屏中提及某人

相较于记忆中的原版插件 加了勿扰模式等功能

目前仅支持中文
# 指令一览
###### 可见图1
| 指令      | 权限 | 用途 |
| --------- | ----- | ----- |
| /at | `N/A` | 可用指令+插件状态查询 |
| /at dnd on/off | `at.dnd (默认就有)` | 勿扰模式(可见下方解释) |
| /at isprefix on/off | `at.admin` | 是否需要输入@才能提及人(可见下方解释) |
| /at enable | `at.admin` | 使能插件功能 |
| /at disable | `at.admin` | 禁用插件功能 |
| /at reload | `at.admin` | 重载插件配置文件 |
# 勿扰模式
当玩家正常使能`/at dnd on`时

玩家将不会收到当其他玩家提及时所发出的提示音与标题

仅会在聊天栏中高亮用户名

此模式可通过`/at dnd off`进行关闭
# isPrefix
当管理员使能`/at dnd on`时
玩家必须要输入@+用户名才可提及某人(见图3)

当管理员使能`/at dnd off`时
玩家仅需输入用户名即可提及某人(见图2)
